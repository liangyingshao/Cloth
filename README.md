# Cloth
## Environment
- ### 1.Visual Studio 2019
- ### 2.Opengl
- ### 3.glm
- ### 4.C++
## EXE Operation Instruction
  - 配置：采用变化步长+ Velocity Verlet算法。
  - 初始画面：球体+四角固定在天花板上的红点紫色布料。
  - 方向键↑↓←→：松开布料四个固定点中的一个。
  - “w”：加入风力。
  - “c”：绘制/不绘制三角形布料。
  - “l”：绘制/不绘制mass之间的弹簧。
  - “s”：绘制/不绘制代表mass的点。
  - “ ”：（空格）恢复初始画面。
  - “T”：切换立方体/球体。
  - Esc：退出。
  - `注：应在英文输入法下使用键盘进行控制。 `
